Circadian Rhythm Device

A Smart Lighting System for Natural Sleep-Wake Cycle Regulation

üéØ Project Overview

The Circadian Rhythm Device is an intelligent lighting system designed to help regulate natural sleep-wake cycles by mimicking the sun's daily light intensity and color temperature patterns. This device addresses the growing problem of circadian rhythm disruption caused by artificial lighting and screen exposure in modern life.

Developed as a personal wellness technology project, the device automatically adjusts its light output throughout the day to support optimal melatonin production and alertness levels. The system features a custom-designed enclosure, precision LED control, and ambient light sensing capabilities.

Complete Circadian Rhythm Device

Fully assembled smart lighting system with WiFi control.

The finished device features a clean, minimalist design with a frosted acrylic diffuser for even light distribution. The ABS enclosure houses dual-channel LED arrays (warm white 2700K and cool white 6500K), ESP32 microcontroller, BH1750 ambient light sensor, and DHT22 temperature/humidity sensor. The compact 180mm √ó 180mm √ó 45mm form factor is optimized for bedside placement while providing sufficient light output for effective circadian rhythm regulation.

SolidWorks Enclosure Design

3D CAD model with optimized thermal management.

The enclosure was designed in SolidWorks with careful attention to light diffusion, heat dissipation, and manufacturability. Key features include: strategic ventilation slots for passive cooling, modular snap-fit assembly for easy maintenance, mounting points for internal electronics, and a two-piece design separating the light diffuser from the base housing. Material selection focused on ABS plastic for the housing (good dimensional stability and machinability) and frosted acrylic for the diffuser (optimal light transmission and diffusion properties).

‚ö° Electronics

Electronics Integration

ESP32-based control system with dual-channel LED drivers.

The electronics design centers around an ESP32 microcontroller providing WiFi connectivity and PWM control. Custom LED driver circuits enable independent control of warm and cool white LED channels with 16-bit resolution for smooth color temperature transitions. The BH1750 ambient light sensor uses I2C communication for environmental monitoring, while the DHT22 provides temperature and humidity data. Power regulation includes a 12V to 5V buck converter for the ESP32 and 3.3V LDO for sensors. All components are laid out on a custom PCB for compact integration and reliable operation.

üî¨ Testing

Light Output Validation

Spectral analysis and lux measurements across color temperature range.

Comprehensive testing validated the device's light output characteristics using calibrated instruments. A lux meter confirmed brightness levels from 50 to 500 lumens across the operating range. Spectral analysis with a spectrometer verified accurate color temperature mixing from 2700K (warm white) to 6500K (cool white) with smooth transitions between settings. The frosted diffuser provides uniform light distribution with minimal hot-spotting. Measurements were taken at multiple distances and angles to ensure consistent performance for typical bedside placement scenarios.

üì± Software

WiFi Control & Mobile App

Custom interface for schedule programming and manual control.

The device connects to home WiFi networks and communicates via MQTT protocol for reliable control. A custom mobile app interface provides intuitive controls for: manual brightness and color temperature adjustment, automated schedule programming based on sunrise/sunset times, customizable sleep and wake routines, ambient light sensor calibration, and real-time device status monitoring. The app includes preset modes for common scenarios like reading, relaxing, and sleeping. All settings are stored in non-volatile memory on the ESP32 to maintain functionality during network outages.

Thermal Management Validation

Long-term temperature testing ensures safe operation and LED longevity.

Thermal testing used infrared imaging and embedded temperature sensors to validate heat dissipation under continuous maximum power operation. Peak temperatures remained well below critical thresholds: LED junction temperature stayed under 70¬∞C (safe for 50,000+ hour lifespan), enclosure external temperature reached only 45¬∞C (safe to touch), and internal component temperatures remained within specifications. The passive cooling design with strategic ventilation slots proved sufficient, eliminating the need for active cooling. Extended 72-hour burn-in testing confirmed thermal stability with no performance degradation or safety concerns.

Research & Concept Development

Circadian Biology Light Spectrum Analysis Human Factors Requirements Definition

üí° Key Insight: Research revealed that blue light (450-480nm) suppresses melatonin production most effectively. This informed the dual-channel LED design using 2700K warm white (minimal blue content) for evening use and 6500K cool white (high blue content) for daytime alertness. Target color temperature range and brightness levels were defined based on published circadian rhythm studies.

Mechanical Design & CAD Modeling

SolidWorks CAD Thermal Analysis Light Diffusion DFM Principles

üí° Key Insight: Initial prototypes had significant hot-spotting and uneven light distribution. Switching to a two-layer diffuser design (frosted acrylic primary + internal reflector) solved this issue. Added ventilation slots after thermal testing showed LEDs exceeding 80¬∞C - the revised passive cooling design maintains junction temperature below 70¬∞C for optimal lifespan.

Electronics Design & Integration

Circuit Design PCB Layout PWM Control Sensor Integration

üí° Key Insight: Early firmware used 8-bit PWM control, resulting in visible flicker at low brightness levels. Upgrading to 16-bit PWM (via ESP32's LEDC peripheral) eliminated flicker completely and enabled imperceptible color temperature transitions. Ambient light sensor placement required iteration - mounting it on the back panel (facing away from LEDs) prevented self-interference while still accurately measuring room lighting conditions.

Firmware Development & Control Algorithms

Arduino Framework MQTT Protocol Time Synchronization Algorithm Development

üí° Key Insight: User feedback revealed that abrupt lighting changes were disruptive and unnatural. Implemented smooth transition algorithms using cubic bezier curves for brightness/temperature changes, making adjustments imperceptible during normal activities. Added adaptive scheduling that automatically adjusts based on sunrise/sunset times for the user's location, eliminating manual seasonal reconfiguration.

Testing, Validation & Refinement

Spectral Analysis Thermal Testing User Studies Safety Validation

üí° Key Insight: Three-week user testing with multiple participants revealed unexpected usage patterns. Many users wanted brighter output for task lighting (reading, working) beyond the initially planned circadian-optimized levels. Added a "boost mode" that temporarily increases brightness while maintaining appropriate color temperature for the time of day. This feature dramatically improved user satisfaction without compromising the core circadian regulation functionality.
